# Homebrew Tap for MLX-Flash

MoE expert caching for Apple Silicon — run AI models too large for your Mac's memory.

## Install

```bash
brew tap szibis/mlx-flash
brew install mlx-flash
```

## Usage

```bash
mlx-flash --port 8080              # Start API server
mlx-flash-chat                      # Interactive chat
mlx-flash-browse                    # See what models fit
```
